{{ name }}

  • v-model directive is used to achieve 2-way binding.

    • Computed Properties:

    Counter : {{ counter }} | secondCounter: {{ secondCounter }}

    Result : {{ result()}} | {{ output }}

  • Problem with using direct method in interpolation is that vue does not understand which methods modifies/uses which properties. (but knows if we use computed properties instead.)
  • So when some other variable (secondCounter) changes,everytime result() will also be unecessarily executed.
  • to avoid this we can use 'computed properties'.(which look like function but act like properties) to cache the result.
