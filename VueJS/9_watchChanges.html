    Alternative to Computed Properties: watch

  • watch is used to execute code upon data changes.
  • watch is used mainly for Asynchronous code, otherwise should prefer computed Properties as they are optimized (but are synchronus).
  • define a property to watch as a key in watch block as shown.
  • Vue automatically passes changed value as argument. (so can catch it.)
